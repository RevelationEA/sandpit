# Revelation Models

This repository mirrors the Revelation EA model catalog referenced by `models.json`,
with each model stored under `reference/models` to keep URLs stable and navigable.

Base URL used for source downloads:
`https://revelationcore.com/beta/`

## Repository structure

```
reference/
  models/
    ArchiMate/
    BPMN-Lite/
    Diagram Types/
    Dynamic Test Suite/
    Getting Started/
    Sector Models/
    Views/
```
